## 普通の電卓(ex02/calc.py)
### 追加機能
- クリア: entryに入力された数字の一文字目のみをdeleteする。
- オールクリア: entryに入力された全ての数字をdeleteする。
- 四則演算: 四則演算を使って計算できる。
- 見栄え: 極力一般に使われている電卓と同じような見た目の電卓を作った。
- 小数点: 小数つけられる。
