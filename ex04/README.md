# 第四回
## 逃げろこうかとん(ex04/dodge_bomb.py)
### ゲーム概要
-  dodge_bomb.pyを実行すると、青空と草原が広がり、こうかとんを動かし、爆弾から逃げる。
-  こうかとんと赤い爆弾があたるとゲームオーバー
### 操作方法
-  矢印キーでこうかとんを上下左右に動かす
### 追加機能
-  爆弾を増やす
-  赤い爆弾を速くする
-  ShiftキーとCtrlキーくを押し間違えるかもしれないから"MISTAKE"とでる
-  青色の爆弾に当たると画像変わる
